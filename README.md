# 一日一笺

一日一笺是一款本地优先的苹果电脑菜单栏记录工具。

隐私为先，你的每一天都值得被看见。

它用于快速记录一天中的两类内容：

- 专注记录：开始、暂停、继续、结束一段专注，并记录实际做了什么。
- 快速记录：随手记下灵感、碎碎念和临时想法。

一天结束后，一日一笺可以把当天记录复制给你选择的 AI 工具，也可以保存为 PDF 方便归档和分享。复制内容仍是结构化文本，适合 AI 读取；PDF 则适合自己查看、转发和留档。

## 当前状态

这是一个早期预览版本。当前仓库可以打包出可双击运行的 `.app`，也可以生成用于下载测试的 `.dmg`。

这个 `.dmg` 已做本地临时签名，但还没有开发者签名和公证。系统首次打开时可能提示无法验证开发者，请按下面的方式打开。

## 下载使用

1. 从发布页面下载 `一日一笺.dmg`。
2. 打开 `.dmg`，把 `一日一笺.app` 拖到“应用程序”。
3. 第一次启动：右键点击 `一日一笺.app`，选择“打开”，然后确认。
   - 也可以到“系统设置”里的“隐私与安全性”中选择“仍要打开”。
4. 启动后，在菜单栏里找到“一日一笺”。它是菜单栏工具，不会显示在程序坞里。

## 本地打包

需要苹果电脑，并安装命令行开发工具。脚本会用到 `swift`、`sips`、`iconutil`、`hdiutil`。

```bash
./scripts/package_app.sh
```

输出文件：

- `dist/一日一笺.app`
- `dist/一日一笺.dmg`

脚本会构建发布版本、根据 `Assets/AppIconSource.png` 生成应用图标、组装 `.app`，并生成 `.dmg`。当前只做本地临时签名，不做开发者签名和公证。`dist/` 已加入忽略列表，只用于本地构建和上传发布附件。

## 功能

- 菜单栏入口。
- 开始、暂停、继续、结束专注记录。
- 快速记录，回车即可保存。
- 今日时间线预览，最新记录在前。
- 已保存记录可修改文字或删除。
- 菜单内可退出应用。
- 本地 JSON 数据持久化。
- 主入口一键复制给 AI：
  - `概览`
  - `时间线`
  - `给人工智能的提示`
- 设置中保存当天 PDF。
- 可配置 PDF 保存目录。
- 不做云同步。
- 当前版本不需要人工智能密钥。
- 不自动上传任何记录。

## 系统要求

- 苹果电脑系统 26.0 或更高版本。
- Swift 6.0 或更高版本。

```bash

```

## 验证

```bash
swift build
swift run DayleafCoreCheck
```

`DayleafCoreCheck` 会验证核心逻辑，包括：

- 专注时长计算，暂停时间不计入；
- 按日期筛选和时间线排序；
- 可配置保存目录；
- 复制给 AI；
- PDF 保存；
- JSON 保存和读取。

## 复制给 AI 和保存 PDF

默认保存目录：

```text
~/Documents/一日一笺/
```

PDF 文件名格式：

```text
YYYY-MM-DD-一日一笺.pdf
```

复制给 AI 的文本结构：

```markdown
# 2026-06-13 一日一笺

## 概览

## 时间线

## 给人工智能的提示
```

文件末尾的提示语是温暖、回顾型的，不是冷冰冰的效率分析。它会引导人工智能像真诚的朋友一样回应你这一天。

## 数据与隐私

一日一笺坚持本地优先。

- 记录只保存在本机 JSON 数据文件里。
- PDF 只保存到你选择的目录。
- 应用本身不会主动上传记录，也不会主动调用人工智能服务分析你的数据。
- 如果你愿意，可以把自己复制出来的记录交给你选择的第三方人工智能模型；这一步由你主动决定，也使用你自己的账号、密钥或服务。
- 我们的承诺是：不替你上传、不替你调用、不拿你的记录做分析。
- 应用不会收集屏幕、键盘、浏览器或其他应用使用记录。

## 已知限制

- `.app` 和 `.dmg` 已做本地临时签名，但没有开发者签名和公证；系统首次打开时仍会提示无法验证开发者。
- 暂无全局快捷键。
- 暂无系统通知。
- 时间线编辑目前只能修改文字，不能修改开始时间、结束时间和时长。
- 当前界面冒烟测试主要靠手动验证。
- 当前本地命令行开发工具环境没有可用 XCTest，所以不使用 `swift test`；核心检查用 `DayleafCoreCheck` 代替。
